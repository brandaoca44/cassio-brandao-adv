# [Cássio Brandão Advocacia](https://cassio-brandao-adv.vercel.app/) 

Site institucional desenvolvido para apresentar os serviços jurídicos do advogado Cássio Brandão, com foco em uma experiência profissional, moderna, responsiva e intuitiva.

## Sobre o projeto

O projeto foi criado para fortalecer a presença digital do escritório, apresentar as áreas de atuação e facilitar o contato com potenciais clientes.

A interface utiliza uma identidade visual baseada em branco, prata e dourado, transmitindo elegância, confiança e sofisticação.

## Funcionalidades

- Página institucional responsiva
- Apresentação profissional
- Áreas de atuação jurídica
- Seção de diferenciais
- Etapas do atendimento
- Formulário de contato
- Integração com WhatsApp
- Menu responsivo
- Botão flutuante de WhatsApp
- SEO básico
- Open Graph para compartilhamento
- Sitemap e robots.txt
- Favicon personalizado
- Dados estruturados com Schema.org

## Áreas de atuação

- Direito dos Contratos
- Direito do Consumidor
- Direito Administrativo
- Direito Trabalhista
- Direito Previdenciário
- Direito Digital
- Consultoria Jurídica
